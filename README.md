# Payment Form App

A payment form that validates credit card information using HTML, CSS, and JavaScript.

## Features

- Credit card number validation using the Luhn algorithm
- Expiration date validation
- CVV validation
- Styled with Water.css classless framework

## Getting Started

Open `index.html` in your browser to run the application.

## Learning Objectives

- Build a payment form with input validation
- Use regular expressions to validate user input
- Apply the Luhn algorithm for credit card validation
- Use array methods like `map` and `reduce`
